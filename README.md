# Audiohosting

### Audiohosting has the following capabilities:
- user registration
- download audio
- listening to your music (audio player with playlist)
- calculation of views, likes, and dislikes

### Requirements:
Django==2.0.5<br>
djangorestframework==3.8.2<br>
Pillow==5.1.0<br>
pytz==2018.4<br>
